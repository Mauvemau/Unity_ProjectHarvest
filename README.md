## A 2D Bullet-Heaven game made in Unity 6.0.42f1

#### **To play the game on editor you must open the boot scene called "Boot"!**

### Controls:
- WASD or Left Stick to move.
- Mouse or Right Stick to aim.
- ESC to close the program.
